Full Stack Developer / Architect (AI Focus)

Lead the architecture and delivery of a new AI-first product at a fast-moving startup.
Full-Time Tech Lead / Architect Hybrid / Remote-Friendly Location: Preferably North America or Europe (±3h from CET/ET)

Company: NovaGrid Labs (early-stage SaaS startup)
Product: Greenfield AI-enabled platform for intelligent workflow automation and insights.

NovaGrid Labs is building an AI-first platform that helps modern teams automate operational workflows, surface insights, and make better decisions using their existing data and tools. We are a small, senior team and are looking for a Full Stack Developer / Architect who can own the technical vision and lead end-to-end implementation of our new AI-focused product.

The Role

You will be the primary technical lead for our new product line: from shaping the architecture and stack to building core services and mentoring future team members. You will work closely with the founders and product to transform ambiguous customer problems into scalable, production-grade solutions.

  • Set the overall system architecture and technical direction for a greenfield AI product.
  • Design and implement full stack features across web UI, APIs, and data layer.
  • Integrate and orchestrate AI/ML capabilities (LLMs, embeddings, vector search, prompt pipelines).
  • Establish engineering best practices (review standards, testing, CI/CD, observability).
  • Collaborate closely with product & design to ship iteratively and validate with customers.
  • Help build and mentor a small engineering team as we grow.

Our Tech Stack

You don't need experience with every item, but you should be comfortable working across similar technologies.

  • Frontend: Angular, React
  • Backend: .NET / ASP.NET Core (C#)
  • Database: SQL Server
  • APIs: RESTful services, JSON, OpenAPI/Swagger
  • AI: LLMs (OpenAI / Azure OpenAI / similar), embeddings, vector search
  • Auth: OAuth2 / OpenID Connect, JWT
  • Cloud: Azure (preferred) or AWS
  • Infra: Docker, container-based deployment
  • CI/CD: GitHub Actions / Azure DevOps (or similar)
  • Monitoring: Application logging, tracing, metrics

What You'll Do

  • Define the architecture for scalable, secure, multi-tenant web applications and services.
  • Design and build Angular or React SPAs communicating with ASP.NET Core backends.
  • Model and optimize SQL Server schemas, queries, indexes, and data access layers.
  • Implement AI-powered features like natural language interfaces, summarization, and recommendations.
  • Design robust API contracts for internal and external integrations.
  • Lead technical decision-making, from framework choices to deployment strategies.
  • Set up DevOps pipelines (build, test, deploy, rollback) and basic observability.
  • Ensure security, performance, and reliability are built-in from day one.
  • Provide technical mentorship and code reviews to raise the bar across the team.

What We're Looking For

  • 6+ years of professional software development experience, with at least 2 years in a senior/lead capacity.
  • Proven experience designing and implementing full stack web applications end-to-end.
  • Strong proficiency in .NET / ASP.NET Core and C#, including experience with layered or clean architecture patterns.
  • Hands-on experience with Angular and/or React building modern, component-based SPAs.
  • Solid understanding of relational databases and SQL Server (schema design, query optimization, transactions).
  • Experience integrating or building on top of AI / LLM APIs (e.g., OpenAI, Azure OpenAI, Hugging Face, etc.).
  • Comfortable with cloud deployment and containerization (Docker) in production environments.
  • Ability to make and communicate architecture trade-offs clearly to technical and non-technical stakeholders.
  • Strong sense of ownership: you are comfortable operating in ambiguity and driving projects to completion.
  • Excellent communication skills and experience collaborating closely with product and design.

Nice to Have

  • Experience in a startup or early-stage environment.
  • Background in enterprise SaaS, B2B data platforms, or workflow/automation tools.
  • Experience with event-driven architectures or message queues (e.g., Azure Service Bus, Kafka, RabbitMQ).
  • Knowledge of domain-driven design (DDD) and clean architecture principles.
  • Experience with role-based access control (RBAC) and multi-tenant SaaS patterns.
  • Exposure to traditional ML, data pipelines, or analytics tooling.
