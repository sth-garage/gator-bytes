Role Overview

As a Junior Front-End Developer, you will work within a cross-functional team to build and maintain front-end features for internal and client-facing web applications built on AngularJS. You’ll collaborate with senior engineers, designers, and product managers while learning best practices for front-end development in an enterprise setting.

What You’ll Do

  • Implement UI components and pages using AngularJS, HTML, and CSS based on design specifications.
  • Work with senior developers to integrate front-end components with RESTful APIs.
  • Fix bugs, perform code cleanup, and help refactor legacy AngularJS code under guidance.
  • Write clear, maintainable JavaScript following team style guidelines and coding standards.
  • Participate in code reviews, sprint planning, and daily stand-up meetings.
  • Assist in creating and updating user-facing documentation and component usage guidelines.
  • Collaborate with QA engineers to reproduce issues and test new features.

What We’re Looking For

  • Some programming background (academic projects, internships, bootcamps, or personal projects).
  • Basic understanding of core web technologies: HTML5, CSS3, and JavaScript.
  • Familiarity with at least one front-end framework; exposure to AngularJS is a strong plus.
  • Ability to read and reason about existing code and make small enhancements safely.
  • Foundational knowledge of version control (Git) and working in a collaborative environment.
  • Curiosity to learn enterprise-scale front-end patterns and best practices.
  • Clear communication skills and willingness to ask questions when unsure.

Nice to Have

  • Side projects or coursework using AngularJS or a similar framework (Angular, React, Vue).
  • Basic understanding of responsive design and cross-browser compatibility.
  • Experience consuming REST APIs from the front end.
  • Exposure to task runners or build tools (npm scripts, Webpack, or similar).

Why Join Our Team

  • Structured onboarding and mentorship from experienced front-end engineers.
  • Clear growth path from junior to mid-level roles within 18–24 months for strong performers.
  • Opportunity to work on high-traffic, production-grade applications used globally.
  • Competitive salary, annual bonus eligibility, and comprehensive benefits package.
  • Hybrid work model with 2–3 days per week in our New York office.
