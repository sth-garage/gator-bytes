Location: Chicago, IL (Hybrid)  |  Department: Engineering
Great opportunity for a junior developer to grow under an experienced engineering team, working primarily with C#/.NET and learning modern development practices.

About Northbridge Software Group

Northbridge Software Group is a mid-sized technology company (150+ employees) that builds line-of-business and analytics applications for clients across finance, logistics, and healthcare. Our engineering team values clean code, mentorship, and continuous learning.

Role Overview

We’re looking for a Junior C# Developer with 1–2 years of professional experience (or equivalent internship/co-op) to join our application development team. You’ll work closely with senior developers, participate in code reviews, and gradually take ownership of features in our core platforms.

The role is primarily focused on back-end and API development in C#/.NET, with opportunities to contribute to front-end development (JavaScript/TypeScript, HTML, CSS) as a plus.

What You’ll Do

  • Implement features and bug fixes in C# using .NET 6+ (or .NET Core).
  • Work on RESTful APIs, data access layers, and background services.
  • Collaborate with senior engineers to design, implement, and refactor application components.
  • Write unit tests and contribute to improving test coverage and code quality.
  • Assist with database queries and schema changes (SQL Server).
  • Participate in code reviews, sprint planning, and daily stand-ups.
  • Help troubleshoot issues across development, test, and production environments.
  • Optionally contribute to front-end components (e.g., React, Angular, or similar) using JavaScript/TypeScript, HTML, and CSS.
  • Document your work and share knowledge with the team.

Required Qualifications

  • 1–2 years of professional experience developing applications in C#.
  • Experience with .NET Core / .NET 5+ or ASP.NET (MVC/Web API).
  • Basic understanding of object-oriented programming and design principles.
  • Familiarity with relational databases (preferably SQL Server) and SQL.
  • Experience using Git or similar version control systems.
  • Comfort working in a team environment and receiving feedback on your code.
  • Strong problem-solving skills and attention to detail.
  • Good written and verbal communication skills.
  • Authorization to work in the United States; we are currently not able to sponsor visas for this role.

Nice-to-Have Skills (Plus, Not Required)

  • Front-end experience with JavaScript/TypeScript, HTML, and CSS.
  • Exposure to a modern front-end framework (e.g., React, Angular, or Vue).
  • Familiarity with Entity Framework or other ORMs.
  • Basic understanding of RESTful API design and JSON.
  • Experience with Azure or another cloud platform.
  • Knowledge of unit testing frameworks (e.g., xUnit, NUnit, MSTest).
  • Experience working in an Agile/Scrum environment.

What We Offer

  • Competitive salary for a junior-level role.
  • Hybrid work environment (2–3 days per week in our Chicago office).
  • Structured onboarding with mentoring from senior developers.
  • Training budget for courses, certifications, and conferences.
  • Comprehensive health, dental, and vision insurance.
  • 401(k) with company match.
  • Generous PTO and paid holidays.
  • Clear growth path toward mid-level Software Engineer roles.
